Getting started with Pingendo

This page explains how to get started

Login

Point your browser to pingendo.com/me
You'll be prompted to login via Github.
If you don't already have a Github account you will be able to create one.
You'll be required to grant Pingendo permissions to work with Github.
For additional info check out the Github website.

Create a project

Once logged in your account page you can click on the "Create a project" button.
Pick a name and choose if you want to make it public or private.

Organize your project

Once a new project is created you'll be redirected to its project page.
Here you can browse project files and create new ones. 
Pingendo initialize new projects with an empty index page, a default theme and a manual page.
So you can just double click on index.html to start editing and building.

Build a page

When you open a page in Pingendo you'll find our visual editor.
It consist of three main areas :
Component panel (at left)
Canvas Right panel
To quickly build a page drag desired sections from the left panel onto canvas.
You can now select an element and modify its properties on the right panel.
To learn more about the editor features take a look at our Guide
