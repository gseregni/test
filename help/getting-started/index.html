How to get started with Pingendo

Learn the basics, to get up and running in 3 minutes.

Login

Point your browser to https://pingendo.com/me
You'll be prompted to login via Github.
If you don't already have a Github account you will be able to create a new one.

In order to work, Pingendo needs to be authorized on your Github account with 'Repositories' privilege -  read more

If you are not comfortable with, we suggest you to create a new Github account dedicated to Pingendo only. Or use the desktop versionof Pingendo that works locally.

Create a new project

Once you’ve set up your Pingendo account it’s time to create your first project, and get started to build.

  • Click on the "Start a new project" button
  • Name your project -- the same name will be used to create a repository on Github to store projects files
  • By default project are set to "Public" but you can also set projects "Private" (Read more about collaboration)

Manage files

Clicking on a project from the account page, will bring you to its dashboard.
Here you can browse project files and create new ones. 
Pingendo initialize new projects with 3 files

  • index.html- an empty page
  • theme.scss- A default Bootstrap 4 theme used by your page 
  • theme.css- CSS compiled version of your theme

Double click on the index.html file, this will open the builder.  

Your first page

Double click on the page  index.html, this will open document.
You can add sections and components from the left panel on canvas.
Once dragged, click on an element to modify its properties with the right panel.
Double click on elements to edit the their content.
