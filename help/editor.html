Work with page builder                             

This page show how to perform most usefull tasks in the page builder.

Basic editing

Edit text

Double click to edit.

Formatting options appear in the top bar.



   

Copy - Undo

Work on all actions

Copy / Paste
ctrl C/V     ⌘ C/V 

Undo / Redo
 ⌘ Z  /  ⌘ ⇧ Z  
 ctrl Z / ctrl ⇧ Z

Double click to pick the image among the uploaded ones.

To link external images, click on image and paste the link in the URL field.
