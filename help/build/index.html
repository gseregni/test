The basics

Let's go hover the basics of UI

  1.  Canvas- Show your document and let you select elements
  2.  Components panel - Let you browse a collection of snippets that you can drag on canvas to  build your pages 
  3. Selection properties - Let you configure the currently select element
  4. Live source editor- 

Components

Once logged on https://pingendo.com/mein your account page you can click on the "Create a project" button.
Pick a name and choose if you want to make it public or private.

Work with files

Clicking on a project from the account page, will bring you to its dashboard.
Here you can browse project files and create new ones. 
Pingendo initialize new projects with 3 files

  • index.html- an empty page
  • theme.scss- A default Bootstrap 4 theme used by your page 
  • theme.css- CSS compiled version of your theme

To start build double click on index.html
