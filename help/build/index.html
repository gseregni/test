Build pages                             

A crash course on how to use Pingendo to build a responsive page with Bootstrap 4 



The basics

Let's go hover the basics of UI.

The workspace is divided in 5 main areas :

  1.  Canvas- Show your document and let you select elements
  2.  Components browser - To browse the available components 
  3. Selection properties - To configure the currently selected element
  4. Live source editor- To inspect and edit HTML and SASS sources

Build

In Pingendo you get two kinds of elements to build your pages 

  • Bootstrap stock components divided in 6 category
  • Premade full-width sections – made of BS4 components – used to quickly shape layouts 

To quickly build prototypes use the left panel – clicking on the lamp – to drag the desired sections and drop them on the canvas.
You'll be only allowed to insert a section after another section, no nesting is allowed.

Once the layout has been built, you can further customize each section adding the desired components. Differently from sections, components can be dragged anywhere in the page. When dropping components the insertion point will be highlighted in blue. 

If you prefer instead to build your layout from scratch, without using premade 
sections, you can still drop directly stock components on the blank canvas.

Configuring components

When select a component, its' properties are shown in the selection panel.
The panel is divided in two parts. The upper one show the component tree structure. The lower one  its properties.

Grid

Bootstrap grid system provides the quick and easy way to create responsive layouts. 
Pingendo  handle mostly automatically the creation of needed grid elements (container, rows and columns).
So while you drop elements on canvas, the request structure will be created.
Every column can be after resized dragging it's left or right border. 
If you don't know how the Bootstrap grid system please have a look at Bootstrap documentation here https://v4-alpha.getbootstrap.com/layout/overview/
