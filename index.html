Perfect for developers but simple enough for everyone

Pingendo is an intuitive and concise HTML authoring app. It's based on Bootstrap, and runs in every modern browser.

Instead of locking you into pre-made templates, Pingendo let you build and iterate on pages freely, faster and with ease.

Works on plain HTML and SASS files for added extendibility and JS integrations, and store them on Github for you and your team.

A collaboration platform backed up by Github to work safely

Github is great, but managing projects and contributing changes on it can be too much for non-techies.                              

Pingendo unify code, content, and design in a single tool and help people in each discipline to collaborate.

We consider it a modern alternative to wireframes and mockup and a valid options to CMS for static sites.
